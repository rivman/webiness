Class WsForm

WsForm Generates HTML form and form widgets.

Example usage:

$form = new WsForm(WsUrl::link('controller', 'form_action_script');

# widgets
$form->textInput(array('name' => 'text_input'));
$form->textInput(array(
    'name' => 'date_input',
    'type' => 'date',
    'placeholder' => 'enter date'
));

$form->show();
